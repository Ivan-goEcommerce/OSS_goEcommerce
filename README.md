# OSS goEcommerce

Eine PySide6-Anwendung für E-Commerce-Suche mit moderner GUI.

## Installation

1. Python 3.12 installieren
2. Abhängigkeiten installieren:
```bash
pip install -r requirements.txt
```

## Ausführung

```bash
python oss.py
```

## Features

- Moderne PySide6 GUI
- Eingabefeld mit Label "Taric Eingeben"
- Such-Button "Suche"
- Dynamische Suchergebnisse
- Threading für nicht-blockierende Suche
- Responsive Design
- Deutsche Benutzeroberfläche

## Projektstruktur

```
oss.py              # Hauptanwendung
requirements.txt    # Python-Abhängigkeiten
README.md          # Dokumentation
```

## Verwendung

1. Starten Sie die Anwendung mit `python oss.py`
2. Geben Sie Ihren Suchbegriff in das Eingabefeld ein
3. Klicken Sie auf "Suche" oder drücken Sie Enter
4. Die Ergebnisse werden im unteren Bereich angezeigt
